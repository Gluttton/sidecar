Detailed Description

Formatting settings for a C++ input or output stream.

Mimics the format control capabilities of std::ios_base + std::ios (for the fill character). Allows one to define and save a specific formatting, and then apply them to values within an output stream.

int main()
{
double d = 1234.56789;
Utils::Format gen4(4);
Utils::Format sci8(gen4);
sci8.scientific().precision(8);
// Should print out the following: 1234.57 1235 1.23456789e+03 1234.57
//
std::cout << d << ' ' << gen4(d) << ' ' << sci8(d) << ' ' << d << '\n';
}
