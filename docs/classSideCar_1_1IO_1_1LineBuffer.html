Detailed Description

Specialization of std::streambuf that reads whole lines of text, counting the number of lines read in.

Useful to use as the buffer for an input stream so that when an error is encountered during processing, some reference where the error occured can be given in an exception.

Overrides setbuf(), overflow(), and underflow() to maintain the line count.
