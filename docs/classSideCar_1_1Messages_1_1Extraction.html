Detailed Description

Data type for a video artifact extraction.

Contains range and azimumth information, and derived X, Y rectangular coordinates. NOTE: the conversion follows the nautical convention that 0 degrees is north, and not the mathematical one (ie x := r * sin(a), not r * cos(a)).
