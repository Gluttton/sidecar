Detailed Description

A general name/value object that holds QVariant values.

Changing the object's value emits the opaqueValueChanged() signal, and possibly the save() signal depending on the method used to invoke the change, restoreValue() or setValue().

The PresetManager class manages one or more Setting objects along with a set of preset values.
