Detailed Description

Documentation for the algorithm Summer.

Generates summed output of N input messages from different sources. The messages used in the summation will all have the same sequence counter value. Also, if an enabled channel has somehow missed a message (a message 'drop'), then there will not be an summed output for that sequence number. In other words, the sequence number must be present on all enabled channels for the algorithm to emit a summed messages.
