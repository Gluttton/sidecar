Detailed Description

Collection of ACE_Message_Block objects received from a network connection.

Defines a Qt meta type for the class so that shared references to instances of the class may be contained in signals sent across threads. A specific instance of this may be found in GUI::ReaderThread which manages a separate thread for reading UDP/TCP data, storing the whole messages in instances of ACE_Message_Block. The mesage blocks are appended to a MessageBlockList, a reference to which is then used in its incoming() signal.

Note that since this object may be shared among threads, it should be treated as a read-only collection once it has been emitted in signal. Also, the MessageBlockList will properly destroy the contained ACE_Message_Block objects when the MessageList instance is destroyed. Therefore, one should always use the ACE_Message_Block::duplicate() method to obtain a personal copy of a held mesage block, for instance when giving it to an IO::MessageManager instance.
