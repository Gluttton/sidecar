Detailed Description

Base class for message-specific circular buffers.

A circular buffer holds a finite number of PRIMessage objects. The value obtained from the message PRIMessageInfo::getShaftEncoding() method determines where in the buffer it is stored.

This class does all of the type-neutral work. However, it prohibits creation of instances. Type-specific classes provide that functionality.
