Detailed Description

Derivation of SettingsBlock that has a BoolSetting object which acts as an 'enabled' setting.

Defines an additional signal, enabledChanged(), which is emitted whenever the enabled state changes. NOTE: enabled state changes do not emit the SettingsBlock::settingsChanged() signal.
