Detailed Description

template<typename T>
class SideCar::Messages::TCircularBuffer< T >

Template class for a type-specific circular buffer.

The template argument must provide a GetMetaTypeInfo() method that returns a reference to a MetaTypeInfo object that describes the messages held in the buffer.
