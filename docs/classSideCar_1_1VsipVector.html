Detailed Description

template<class T, class DatumType = typename T::value_type>
class SideCar::VsipVector< T, DatumType >

Provides a simple wrapper around std::vector-like data structures.

T needs to have T.size() and T[0] operators. DatumType needs to match the size of entries in T.
