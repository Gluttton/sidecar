Detailed Description

Represents a collection of TestObj objects.

Extends TestObj interface to register one or more unit tests. Executing Suite::run() exercises all of them, in the order that they were registered. Because Suite only relies on the TestObj interface, nesting of Suite objects (and other container types) is possible. Here's a simple example:

struct Foobar : public UnitTest::TestObj
{
void test() { assertEqual(2, 2); }
};
int
main()
{
UnitTest::Suite s("Suite");
s.add(new Foobar);
return s.mainRun();
}
