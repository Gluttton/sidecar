Detailed Description

Utility functor that inserts a textual representation of a Header's header data into a std::ostream object.

Example of it use:

LOGDEBUG << msg.headerPrinter() << std::endl;

Relies on Header::printHeader() to do the actual conversion of header information to text, which derived classes may override.
