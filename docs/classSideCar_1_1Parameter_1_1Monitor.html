Detailed Description

template<typename Parameter>
class SideCar::Parameter::Monitor< Parameter >

Monitor and apply changes to a Value instance.

The setValue() method will apply a change to the associated Value instance, and will invoke the valueChanged() notification method, which derived classes must define.
