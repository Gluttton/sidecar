Detailed Description

Terminates active 'runner' and 'tail' processes on a set of hosts that are running under the current user's ID.

In normal operation, the SideCar system properly starts and stops remote processes. However, in abnormal circumstances, a process may fail to terminate properly, in which case a Cleaner instance will do the job, indiscriminately and with gusto.
