Detailed Description

Base class for SideCar message readers used by the GUI applications.

Contains no actual connection code; see derived classes TCPMessageReader and MulticastMessageReader.

Manages two MessageList objects, one that is active for incoming data, and one that was active but can safely be used by another thread to obtain received data.
