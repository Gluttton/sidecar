Detailed Description

Interface that all sampling message implementations must support.

Provides type-independent information specific to the sampling event: high-frequency counter, PRI counter, shaft encoding value, PRF encoding value, IRIG time, range constants for linear conversion from sample offset to range, and the number of samples.

Derived type-specific sampling messages provide access to the actual sample values.
