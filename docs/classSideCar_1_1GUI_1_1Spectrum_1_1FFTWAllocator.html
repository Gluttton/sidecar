Detailed Description

template<typename T>
class SideCar::GUI::Spectrum::FFTWAllocator< T >

STL allocator implementation that uses the FFTW library for low-level memory allocation.

The FFTW routines guarantee proper alignment for SSE and Altivec array processing.
