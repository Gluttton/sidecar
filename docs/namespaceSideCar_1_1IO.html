Detailed Description

Namespace for entities that handle SideCar data I/O.

The SideCar system provides a publisher/subscriber infrastructure that supports auto-discovery of services using Apple's Zeroconf system (see the documention of the SideCar::Zeroconf namespace for additional details).

The ACE classes handle nearly all of the low-level I/O details, providing a nice abstraction layer for the SideCar classes. The SideCar system provides file and socket I/O source and sink end points. Interfacing with the ACE infrastructure occurs in the IO/Readers.h and IO/Writers.h files.
