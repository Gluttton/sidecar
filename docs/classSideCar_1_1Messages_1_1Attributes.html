Detailed Description

Collection of name/value pairs for SideCar messages.

Relies on the XMLRPC library for marshalling the values to/from network or disk. Currently, the Extraction and BugPlot messages support attributes. If enough messsage classes do, it may make sense to have Header acquire an Attributes slot.
