Detailed Description

Contains information obtained from resolving a ServiceEntry object.

In the Zeroconf world, services found by a Browser do not reveal concrete connection information, they only contain a name and a type. Client applications of a Browser must ask a ServiceEntry for its connection information via ServiceEntry::resolve(). This queries the multicast DNS server running locally, which returns the host/port connection pair.
