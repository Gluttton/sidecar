Detailed Description

Container of HistoryFrame objects that represent snapshots of past data.

The size of the container is bounded by a duration parameter, which limits how old the oldest record in the buffer may be. Does not delete old frames, but rather places them on a free list. For near-constant data rates, this should allow the HistoryBuffer to reach a stable point where it no longer allocates new frames.
