Detailed Description

General color lookup table class.

Instances provide via getColor(double) a mapping from intensity values in the range [0.0-1.0] into RGB values stored in a Color object. There are kNumTypes predefined tables with different color ramps. The setType() method selects which table a CLUT object uses.
