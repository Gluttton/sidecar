Detailed Description

Base class for all PRI-type messages created in the SideCar.

Each message contains header information from the VME system, followed by the data samples.

Previously, PRIMessage held a shared reference to a PRIMessageInfo object, which detailed radar configuration parameters. This is no longer the case. Radar settings are now contained in the RadarConfig class.
