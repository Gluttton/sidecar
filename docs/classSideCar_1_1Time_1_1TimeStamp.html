TimeStamp TimeStamp::ParseSpecification ( const std::string &  spec,
const TimeStamp &  zero 
)
static

Parse a time specification and return the TimeStamp value it represents.

Empty and invalid specifications throw InvalidSpecification errors. Supports relative and absolute specifications. Absolute times contains seconds and microseconds integer values, separated by a colon (':') character. Relative times start with a plus ('+') character followed by the number of seconds in the offset. This may be specified as a floating-point number, or as minutes ':' seconds. The resulting offset value is then added to the given 'zero' value and and returned.
