Detailed Description

Specialization of FilePath that used the environment variable $AP_CONFIG_DIR to locate configuration files.

ConfigFilePath first looks for a file in the location given by the FilePath::pathPart() method. If it is not there, try using the value of $AP_CONFIG_DIR and looking there.

NOTE: the 'AP' of AP_CONFIG_DIR stands for "airport". This class was used by the Runway Status Lights project (RWSL). See http://rwsl.ll.mit.edu/index.html
