Detailed Description

Collection of render attributes associated with a particular ChannelConnection object.

Contains state and cached data to improve rendering times when data from a channel does not change.

A Visualizer widget may subscribe to one or more input channels. To keep rendering as fast as possible, input data is first converted into render data, and then at the next Visualizer update, it draws the render data of all of the input channels in one shot.
