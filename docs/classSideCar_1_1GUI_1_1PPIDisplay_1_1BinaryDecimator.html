Detailed Description

Simple data decimator for SideCar BinaryVideo messages.

Decimates by returning a true value if any of the samples within the span defined by the decimation factor is true. A decimation factor of 1 results in no decimation. A decimation factor of 2 will return the maximum of every two samples.
