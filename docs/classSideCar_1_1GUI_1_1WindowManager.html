Detailed Description

Manager of an application's 'Window' menu.

The 'Window' menu contains QAction objects that affect the size and visibility of MainWindowBase objects. It also contains a list of the active MainWindowBase objects, and automatically manages shortcut keys for them. Finally, the 'Window' menu contains a submenu that lists all of the active ToolBar objects.
