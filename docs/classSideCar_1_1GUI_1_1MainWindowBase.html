Detailed Description

Common base class for all SideCar GUI projects.

Holds features and functionality shared among all of the GUI applications. Saves and restores the window position and size between application runs. Manages window minimization, mazimization, and full-screen mode using a common window menu, managed by the application's WindowManager object.
