Detailed Description

Simple averaging filter for azimuth deltas.

Maintains a running average of N (2000 default) beam widths in order to obtain a slowly varying beam width value. Also provides the average number of radials one would expect to find in a complete rotation of the radar. The expectation when using this filter is that the angular velocity of the radar remains fairly constant, even when subjected to wind gusts.
