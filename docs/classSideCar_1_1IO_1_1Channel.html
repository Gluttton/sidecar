Detailed Description

Representation of a named and typed data channel.

Channels represent typed-connections between IO::Task objects. Channels are one-way, from sender to one or more receipients. Use addRecipient() to add a new connection to another IO::Task object.

All recipient connections have an index value associated with them, which is the index into the recipient's list of input channels. This value is conveyed to the receiving Task by storing it in the priority slot of the ACE_Message_Block of the containing the message. The code for RecipientList::deliver() describes this in detail.
