Detailed Description

Extension of the Messages::Extraction class that plots for extractions and user picks.

Contains a QTime attribute that holds the time when the TargetPlot was born. The History class uses this value to determine when to delete a TargetPlot object. The PPIWidget class uses a normalized version of this value to determine the alpha value used to draw the TargetPlot graphical representation: older TargetPlot objects have a lower alpha value, and thus appear fainter than those that are younger.
