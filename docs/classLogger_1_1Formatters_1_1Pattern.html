Detailed Description

Concrete Formatter class that uses a formatting pattern (ala printf and friends) to determine what components to put in the formatted output.

A pattern consists of a text string, with one or more formatting flags. A formatting flag starts with a % character The following formatting flags may be used in a formatting pattern:

  • % insert a `' character
  • c insert the log message category value
  • m insert the log message text
  • M insert the timestamp's microseconds value
  • p insert the log message priority name
  • P insert the log message priority value
  • S insert the timestamp's seconds value
  • w insert the timestamp of the log message. This uses the formatting defined by the TimeVal::operator<<() method for std::ostream streams.
  • W generate a text representation of the timestamp using a format similar to that of the Unix date utility
  • ' take the formatting text between two single quotes and use as the format string for the strftime C library call. See the man page for strftime(3C).
  • z insert a '
    ' character
