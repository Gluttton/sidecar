Detailed Description

Representation of an FFT processing step.

Accumulates input data that will be used for an FFT, as well as the results of an FFT calculation.

WorkRequest::WorkRequest ( const WeightWindow &  weightWindow)

Constructor for the first or 'master' WorkRequest object.

This object will create a FFTW3 plan for use by all other WorkRequest objects to efficiently perform their FFT calculations.

Parameters
