Detailed Description

Configuration file loader.

Reads in a SideCar configuration file, and parses <radar> and <runner> entity tags. For the former, it parses the <radar> children and initializes a Messages::RadarConfig instance with the children values. For the <runner> entities, it creates an instance of the RunnerConfig class to store the <runner> attributes and to hold the XML DOM node that contains one or more <stream> entities.

Uses the Qt XML library (QtXml) to parse the whole configuration file.
