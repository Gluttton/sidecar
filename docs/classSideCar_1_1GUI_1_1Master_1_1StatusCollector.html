Detailed Description

Status collector for Runner StatusEmitter objects.

Accepts XML status information from a StatusEmitter that details the status of the IO::Stream and IO::Task objects owned by the Runner. Hands off incoming status data to a ServiceModel object via ServicesModel::updateStatus() method.
