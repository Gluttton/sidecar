Detailed Description

template<typename TypeTraits>
struct SideCar::Parameter::Defs::BasicRangedTypeDef< TypeTraits >

Variant of the BasicTypeDef template that defines parameter values with a valid range.

The TypeTraits parameter must provide everything described in the BasicTypeDef commentary as well as the following:

  • GetMinValue: method that returns the lower bounds of the valid range
  • GetMaxValue: method that returns the upper bounds of the valid range
