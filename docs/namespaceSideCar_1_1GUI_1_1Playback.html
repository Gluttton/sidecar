Detailed Description

Namespace for the Playback application.

The SideCar Playback application can read binary files created by SideCar algorithms when they had their recorders turned on. Unlike the PRIEmitter application, Playback can retransmit data from multiple files at the same time. It also supports random access within the recording files, jumping forwards and backwards in time using both relative and absolute times.
