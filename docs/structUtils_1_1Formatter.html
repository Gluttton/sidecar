Detailed Description

template<class T>
struct Utils::Formatter< T >

Helper class that binds a value to a Format object for use with standard C++ output streams.

Instances of this class are generated by the Format::operator(), and its attributes are used by the ostream inserter template function for Formatter objects.

The idea for this and most of the framework were shamelessly taken from "C++ Programming Language (3rd Edition)" by Bjarne Stroustrup (pp 635-636)
