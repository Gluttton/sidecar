Detailed Description

Specialization of the PoolAllocator for ACE_Message_Block objects.

There is a separate allocator for the ACE_Data_Block object held internally by an ACE_Message_Block. Grants friend access to an ACE_Singleton template class in order to allow it to create and manage a singleton instance of this class.
