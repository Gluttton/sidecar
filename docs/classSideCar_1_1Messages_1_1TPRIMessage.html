Detailed Description

template<typename _V>
class SideCar::Messages::TPRIMessage< _V >

Templated version of the PRIMessage class.

Contains type-specific implementations and overrides. The sole template parameter is a type trait definition structure, which must contain the following definitions:

  • Type – data type for a single sample (eg. int16, float)
  • Reader – function to call to read in sample data from a CDR input stream
  • Writer – function to call to write sample data to a CDR output stream
  • Printer – function to call to print sample data to a C++ text stream

Since nearly all of the type-specific functionality is contained in the type traits structure, TPRIMessage classes are very light-weight in terms of code bloat.
