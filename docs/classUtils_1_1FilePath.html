Detailed Description

Simple wrapper around the std string class that can decompose a file path into three parts: directory path, file name, and extension.

It will also expand embedded shell variables such as `$FOO' or `${HOME}'.

Returns the extension found at the end of the file name.

An extension is defined as any characters following the last `.' found in the file path. If there are more than one `.' in the path, only the last one is treated as the extension.
