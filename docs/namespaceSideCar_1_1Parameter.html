Detailed Description

Definitions for algorithm runtime parameter settings.

A runtime parameter is initialized with a value when an algorithm starts up, and it may be changed while an algorithm is running via XML/RPC.

Parameter objects support reading and writting to/from files and sockets, in both binary and XML format.
