void VertexColorArray::checkCapacity ( size_t  request)

Check if the given request can be handled by the current capacity of the array.

If expansion is perfomed, this will invoke glVertexPointer and glColorPointer to use the new pointer values. NOTE: because of this side-effect, this method must be called inside a valid OpenGL context.
