Detailed Description

The input side of a publisher/subscriber pair.

Manages a separate thread that hosts a TCP/UDP reader which obtains data from the publisher. Uses the incoming() signal to announce the arrival of new messages.

The useServiceEntry() method gives the subscriber connection information of a publichser to connect to.
