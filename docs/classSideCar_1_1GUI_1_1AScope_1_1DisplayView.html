Detailed Description

Container that hosts a Visualizer widget.

Uses the QFrame base class to show an indicator (frame) when the held Visualizer object has focus. Contains two ScaleWidget objects that provide horizontal and vertical scales next to the Visualizer object. Provides the Visualizer with its background QImage that shows the grid lines created by the ScaleWidget objects.
