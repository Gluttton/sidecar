Detailed Description

Item delegate for the "Name" column.

Variant of QStyledItemDelegate used to show loading progress for an Emitter.

Takes into account installed indexWidget() widgets when calculating sizeHint() values.

Installed in the files_ QTableView for the FileModel::kName column. Draws a progress bar in the background that shows how much of the PRI file the Emitter object has processed during its load.
