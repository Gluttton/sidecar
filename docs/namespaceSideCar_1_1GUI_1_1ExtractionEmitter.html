Detailed Description

Namespace for the ExtractionEmitter application.

The ExtractionEmitter application emits Messages::Extractions messages. It was originally created to test extraction plotting in the GUI::PPIDisplay application before the Algorithms::Extract algorithm was available.

Now that there is a ready source of extractions from that algorithm, this application should probably be retired and deleted from the code tree.
