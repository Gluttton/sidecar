Detailed Description

Concrete implementation of the Monitor class that relies on the ACE infrastructure to determine when a Zeroconf object has data available for processing.

When there is, it invokes Transaction::processConnection() to handle the incoming data.

The handle_input() method is the notification mechanism that ACE uses when there is data available from the file descriptor or socket managed by the ACEMonitor.
