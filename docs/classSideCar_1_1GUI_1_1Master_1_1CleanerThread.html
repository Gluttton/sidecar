Detailed Description

Worker thread used by the Cleaner class to perform the cleaning actions.

Invokes a remote 'killall' command on each host given in the constructor that looks for and terminates 'runner' and 'tail' processes running under the user's login ID.

Emits the finishedHost() signal after each 'killall' command completes, and a finished() signal when it has finished processing all hosts.
