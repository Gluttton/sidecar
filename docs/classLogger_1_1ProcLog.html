Detailed Description

Utility class used to create log messages prefixed with a procedure name.

To use, simply create an instance and use in place of a Log instance. To minimize routine entry times, one can create a static instance like so:

void Foo::bar() {
static Logger::ProcLog log("bar", Log());
log.debug() << "hi mom!" << std::endl;

The above assumes that the class Foo defined a (class) method called Log() which returned the general log device to use for all Foo-related log messages.
