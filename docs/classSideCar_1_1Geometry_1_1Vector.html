Detailed Description

Tuple representing a vector in 3-D space.

Supports basic vector/vector and vector/scalar operations. Borrowed from RWSL project. Calculates on demand magnitude and direction values.

NOTE: not sure if elevation and Z axis are handled properly. Need to be vetted.
