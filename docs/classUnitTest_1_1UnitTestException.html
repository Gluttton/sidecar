Detailed Description

Exception class thrown by a failed assertion in a unit test.

Records the failure message, the file it was found in, the line of the file, and the name of the test that failed the assertion. For the UnitTest framework, there are two classifications of exceptions: those thrown by a failed assertion (assertTrue() or assertEQ()), and all others. The first are classified as test failures, while the latter are run-time errors (eg. file not found). Both classifications are represented by UnitTestException (perhaps mistakenly so). However, only test failures report file and line information.
