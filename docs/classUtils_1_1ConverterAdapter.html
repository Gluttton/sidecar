Detailed Description

template<typename _Iterator, typename _Converter>
class Utils::ConverterAdapter< _Iterator, _Converter >

Adapter for an iterator that performs some type of conversion for each value obtained from a forward iterator.

The first template argument identifies the type of the iterator used to fetch values. The second is a functor that must define an operator() method which does the conversion from one type to another.
