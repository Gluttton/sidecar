Detailed Description

Utility functor that inserts an XML representation of message data into a std::ostream object.

Example of it use:

LOGDEBUG << msg.xmlPrinter() << std::endl;

Relies on Header::printData() to do the actual conversion of data to text, which derived classes may override.
