Detailed Description

Define a locking strategy for the message blocks allocated by a MessageManager.

The SideCar messaging system uses ACE message queues to hold ACE_Message_Block objects. This locking strategy uses a mutex to protect ACE_Message_Block internals when they exist in a multithreaded environment.
