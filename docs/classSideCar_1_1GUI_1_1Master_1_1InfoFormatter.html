Detailed Description

The InfoFormatter class represents a formatter for the 'info' status field of Algorithm/Controller objects.

One cannot create InfoFormatter objects directly, rather one must call the Find() class method in order to obtain an InfoFormatter object for a given name. If the object does not exist, the Find() method will create a new one and register it with the class formatters_ attribute, Otherwise, Find() returns the already-registered object.
