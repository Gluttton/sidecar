Detailed Description

Variant of Qt QXmlStreamReader class that encapsulates some common SideCar uses.

Also solves a linking problem on MacOS X due to some crufty symbol renaming going on by Trolltech/Nokia (I don't think they'll be doing THAT again, given all of the complaints this caused...)
