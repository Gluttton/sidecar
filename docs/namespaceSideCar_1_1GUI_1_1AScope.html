Detailed Description

Namespace for the AScope application.

The AScope application is a simple software simulation of the venerable oscilloscope. It accepts incoming data from one or more channels containing Messages::Video data. The application supports multiple windows, each with one or more independent views of the incoming data. It also provides zooming and panning of the display view.
