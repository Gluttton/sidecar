Detailed Description

A modal dialog that shows a collection of editor widgets with runtime parameter values for an algorithm.

In addition to the parameter editor widgets, the dialog contains four action buttons: - OK – accept parameter changes, and close the editor - Apply – accept parameter changes, but keep the editor open - Undo – undo any changes to the parameter editors, and keep the editor open. - Original – revert any changes to the parameter editors to their original values and keep the editor open. - Close – close the editor, ignoring any changes.
