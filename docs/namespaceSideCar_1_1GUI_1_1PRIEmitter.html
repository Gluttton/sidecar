Detailed Description

Namespace for the PRIEmitter application.

The PRIEmitter application emits data from a file at a fixed frequency. Supports sending data over TCP and UDP (multicast) transports. Note that much of the functionality of this application has been transplanted to the SideCar::GUI::Playback utility, which supports emission of data from more than one file at a time.

It is kept around for sentimental reasons – it was the first SideCar Qt application.
