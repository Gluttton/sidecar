Detailed Description

A wrapper for the Utils::Pool class which allows it to be used as an ACE memory allocator.

The Utils::Pool class is a source of same-size memory objects with fast creation/destruction times. This class serves as the base class for the MessageBlockAllocatorImpl, DataBlockAllocatorImpl, and MetaDataAllocatorImpl classes below.
