int TCPInputHandler::handle_input ( ACE_HANDLE  device = ACE_INVALID_HANDLE)

Gives its SocketReader object a chance to fetch data from the socket, and if there is a finished message available, it will pass it in a call to ServerSocketReaderTask::put().

NOTE: the return of 1 when successful tells the ACE_Reactor object to revisit this handler before doing another ::select call. Override of ACE_Event_Handler method.
