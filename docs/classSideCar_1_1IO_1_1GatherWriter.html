Detailed Description

Collector of CDR-encoded SideCar messages for writing to a IO::Writer device once some limit has been reached.

Contains two parameters that control how many messages it collects before ending to the IO::Writer device:

  • sizeLimit – number of bytes to collect before sending
  • countLimit – number of messages to collect before sending

If neither limit is set, or both are set to zero, then the collector will simply write each message as it receives them.
