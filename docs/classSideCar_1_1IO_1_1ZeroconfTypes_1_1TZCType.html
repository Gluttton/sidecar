Detailed Description

template<ZeroconfRegistry::ID id>
class SideCar::IO::ZeroconfTypes::TZCType< id >

Template class that adds class methods that return character types for the template parameter type ID as well as for its twin ID value.

Other classes may derive from this template in order to assign a Zeroconf type, and to get methods to access the types.

Obtain the "twin" Zeroconf type for objects derived from this class.

A "twin" is the counterpart of a publisher/subscriber pair. If the template class was defined to be a publisher, then this class method should return the appropriate subscriber type.
