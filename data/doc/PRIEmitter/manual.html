PRIEmitter Application

The PRIEmitter application emits SideCar messages held in a data file. Unlike the Playback application, the PRIEmitter application only works with one data file at a time. However, it is still useful for network troubleshooting and for application testing. It is of historical note as it was the very first GUI application written for the 331 SideCar system.
